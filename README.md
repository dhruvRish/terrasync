# TerraSync

Smart salinity monitoring for Oman's Al-Batinah coast. Combines Arduino soil sensors with Google Earth Engine satellite analysis (NDSI/NDVI/SI) to detect and map saltwater intrusion in real time. Data flows through Google Sheets to a live dashboard at [dhruvRish.github.io/terrasync](https://dhruvRish.github.io/terrasync).
